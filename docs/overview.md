# Stock Direction Trainer Overview

The Stock Direction Trainer is a Next.js 14 web app that helps traders practice anticipating the next day's price move using historical candlestick data. The experience combines lightweight charts, responsive controls, and persistent statistics so users can iterate on their intuition.

Key highlights:
- Built with TypeScript, Tailwind CSS, and the Next.js App Router.
- Fetches recent Yahoo Finance data with a bundled CSV fallback for offline play.
- Tracks results locally to surface accuracy metrics and mistake reviews.
